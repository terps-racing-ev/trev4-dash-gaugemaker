# trev4-dash-gaugemaker
Web tool to generate guage data JSON files


<h3>Label / Common Variables </h3>

- Label (string)

<h3>Gauge Common Variables </h3>

- Signal Name (string)
- Minimum Value (number)
- Maximum Value (number)
- Decimal Places (number)
- X Position (number)
- Y Position (number)
- Width (number)
- Height (number)
- Box Color (rgb color)
- Border Color (rgb color)
- Text Color (rgb color)

<h3>Signed Linear Gauge</h3>

- Positive Color (rgb color)
- Negative Color (rgb color)
- Vertical (boolean)
- Show Value (boolean)
